# Requirements

Implement Nx monorepo support for Jet, including:
1. Workspace detection to identify Nx monorepos.
2. Integration with the Nx project graph to understand dependencies.
3. A task pipeline for executing 'jet build' and 'jet install' as Nx targets.
4. Correct handling of Nx-specific configurations (nx.json, project.json).
