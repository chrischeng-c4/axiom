# Requirements

Expand HTML and CSS support with symbol builders and additional lint rules.

1. HTML symbol builder: extract tag IDs, classes, form names, anchor hrefs, meta tags, script/link sources
2. CSS symbol builder: extract selectors, class names, ID selectors, custom properties (--var), @keyframes names, @media queries
3. HTML lint expansion to 10+ rules: missing alt on img, missing lang on html, deprecated tags, empty href, missing meta charset, form without action, duplicate IDs, inline styles, missing title, script without type
4. CSS lint expansion to 10+ rules: add vendor prefix warnings, shorthand property optimization, color format consistency, zero-unit removal, font-face missing descriptors, media query best practices
